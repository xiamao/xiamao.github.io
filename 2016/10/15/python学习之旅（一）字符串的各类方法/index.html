python学习之旅--String各种操作方法

发表于 2016-10-15   |   分类于 学习   |  

本节主要介绍我在学习python过程中遇到的一些对字符串的各类操作，主要有字符串长度函数len()，删除字符串首尾特殊字符的函数strip()，拼接字符串join()函数等。

1、len()函数

len(str),用来获取字符串长度，返回int值。

代码： 

str = 'xiamao'
str_len = len(str)
print 'str_len=',str_len

运行结果：
strlen= 6

2、字符串连接

字符串的简单连接可以通过’+’来实现，’+’ 可以连接多个字符串变量或者字符常量串代码如下：

str1 = 'xia'
str2 = 'mao'
str3 = str1 + str2 + '!'
print 'str3:',str3

输出结果：
str3: xiamao!

3、strip()函数

str.strip()删除字符串首尾的特殊字符，如空格，’\r\n’，’\t’等；
str.strip(sstr)删除字符串首尾包含在sstr中的字符；
str.lstrip()仅删除字符串首部的特殊字符；
str.lstrip(sstr)删除字符串首部包含在str中的字符；
str.rstrip()仅删除字符串尾部的特殊字符；
str.lstrip(sstr)删除字符串尾部包含在str中的字符；
以上各方法的返回值仍然是一个字符串。

代码：

>>> str = 'xia,mao'
>>> str.strip(',')
'xia,mao'
>>> str.strip('ixa')
',mao'
>>> str = ' xiamao '
>>> str.strip()
'xiamao'
>>> str = ' xiamao '
>>> str.lstrip()
'xiamao '
>>> str.rstrip()
' xiamao'

4、join()函数

join()函数功能很多，不仅仅对字符串奏效，同样可以使用于列表、元组、字典等。由于本节主题是对于字符串的操作，因此本节暂时只介绍join()函数在字符串中的应用。

join()函数具体使用，’sp’.join(str)意即将sstr插入到str的每个字符之间。其中sp为分隔符，可以为空。

代码：

>>> str = 'xiamao'
>>> ' '.join(str)
'x i a m a o'
>>> '/'.join(str)
'x/i/a/m/a/o'

5、split()函数

刚刚介绍了join()函数，就不得不提一下与之相对的split()函数，split()函数的作用与join恰好相反，str.split(‘sp’)的功能是，已sp位分隔符将str进行切分。

代码：

>>> str = 'xiamao'
>>> ':'.join(str)
'x:i:a:m:a:o'
>>> str2 = ':'.join(str)
>>> str3 = str2.split(':')
>>> print str3
['x', 'i', 'a', 'm', 'a', 'o']

6、字符串的截取

有时我们会需要只取某个字符串的其中某个小字符串，因此我们就需要字符串的截取功能。用法是str[a,b]，其中a、b即为要截取的其实位和终止位，但是这里是包含a但是不包含b的，又称包左不包右；另外，a取0时可以省略，b可以取赋值，-1代表最后一位。想取到最后一位的方法是，忽略不写。

代码：

>>> str = 'xiamao'
>>> str[0:2]
'xi'
>>> str[:]
'xiamao'

7、lower()函数和upper()函数

在工作中，需要将字符串全部转换位大写或者小写，开始想到的是直接对其assic码进行操作，感觉特麻烦，后来发现了python中自带的可以将字符串转成大写或小写的函数，使用起来很方便。

代码：

>>> str = 'xiamao'
>>> str.upper()
'XIAMAO'
>>> str2 = str.upper()
>>> str2.lower()
'xiamao'
